# What is this?

It is a Web UI for file management with a goofy name.

You can manage the following:

- Roms
- Saves
- Save States (wip)
- BIOS
- Box Art
- Wallpapers

---

# How do I use it?

1. Open settings and pick `Nonna's Kitchen`.
2. Scan the QR Code or enter the URL into a browser.
3. Enter the PIN shown on screen.
4. Manage your files.
